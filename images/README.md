# Program Unggulan DPC PKS Ciksel

Dokumentasi visual program-program unggulan yang dijalankan oleh DPC PKS Ciksel.

## Program Unggulan

### 1. Klinik Gratis PKS
**Kategori:** KESEHATAN

Layanan kesehatan dasar gratis setiap akhir pekan untuk masyarakat prasejahtera.

![Klinik Gratis PKS](klinik-gratis-pks.jpg)

Kegiatan ini menunjukkan petugas medis PKS memberikan pemeriksaan kesehatan dan layanan medis kepada masyarakat secara gratis. Program ini memastikan akses kesehatan dasar untuk semua kalangan masyarakat.

---

### 2. Beasiswa Pendidikan
**Kategori:** PENDIDIKAN

Bantuan biaya pendidikan pelajar berprestasi melalui program KIP Kuliah dan PKRM Afirmasi.

![Beasiswa Pendidikan](beasiswa-pendidikan.jpg)

Program beasiswa formal yang diberikan kepada pelajar berprestasi. Foto menunjukkan acara penyerahan beasiswa oleh pejabat dan institusi pendidikan kepada penerima manfaat. Ini adalah bentuk komitmen dalam mendukung pendidikan generasi muda.

---

### 3. Bedah Rumah Warga
**Kategori:** SOSIAL

Renovasi rumah tidak layak huni bagi warga prasejahtera.

![Bedah Rumah Warga](bedah-rumah-warga.jpg)

Kegiatan solidaritas sosial yang melibatkan rombongan pejabat, relawan, dan masyarakat untuk melakukan perbaikan dan renovasi rumah-rumah tidak layak huni. Program ini memberikan bantuan langsung kepada keluarga prasejahtera untuk meningkatkan kualitas hidup mereka.
